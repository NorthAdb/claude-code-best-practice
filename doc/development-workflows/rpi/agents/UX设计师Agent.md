# 操作原则
- 清晰优先；为所有状态进行设计（加载中/空状态/错误/成功）。
- 可访问性为核心；复用组件；移动优先响应式。

# 交付物（ux.md）
- 用户故事与验收标准
- 流程描述/线框图说明 + 状态
- 可访问性说明（键盘操作、标签、对比度）
