# 能力强化（Power-ups）最佳实践

## 什么是 Power-ups？

Power-ups 是 Claude Code 内置的**交互式功能教学**。每个 Power-up 通过生动的动画演示教会你一个 Claude Code 的特性。

**启动方式：** 在 Claude Code 中输入 `/powerup`

---

## 10 个 Power-up 课程

| # | 课程名 | 主题 |
|---|--------|------|
| 1 | **与代码库对话** | `@` 文件引用、行引用 |
| 2 | **用模式掌控** | Shift+Tab 切换 Plan/Auto 模式 |
| 3 | **撤销一切** | `/rewind`、双击 Esc |
| 4 | **后台运行** | 任务管理、`/tasks` |
| 5 | **教 Claude 你的规则** | `CLAUDE.md`、`/memory` |
| 6 | **用工具扩展** | MCP、`/mcp` |
| 7 | **自动化工作流** | Skills、Hooks |
| 8 | **复制你自己** | Subagents、`/agents` |
| 9 | **随处编码** | `/remote-control`、`/teleport` |
| 10 | **调节模型** | `/model`、`/effort` |

---

## 学习建议

1. **新用户**：从头到尾过一遍 10 个课程
2. **老用户**：挑不熟悉的课程重点学习
3. **团队培训**：让新成员从 Power-ups 开始
