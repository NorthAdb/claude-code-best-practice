## 什么是 Power-ups？

Power-ups 是 Claude Code 内置的**交互式功能教学**。每个 Power-up 通过生动的动画演示教会你一个 Claude Code 的特性。

**启动方式：** 在 Claude Code 中输入 `/powerup`
