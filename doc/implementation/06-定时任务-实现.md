`/loop` 技能用于按 cron 间隔调度重复任务。下面是 `/loop 1m "tell current time"` 的演示——一个每分钟触发一次的简单循环任务。

---

## Loop 演示

### 1. 调度任务

<p align="center">
  <img src="../../implementation/assets/impl-loop-1.png" alt="/loop 1m tell current time——调度和 cron 设置" width="100%">
</p>

`/loop 1m "tell current time"` 解析时间间隔（`1m` → 每分钟一次），创建 cron 作业，并确认调度。关键点：

- Cron 的最小粒度是 **1 分钟** — `1m` 映射为 `*/1 * * * *`
- 重复任务 **3 天后自动过期**
- 作业是**会话范围的**——只在内存中存活，Claude 退出时停止
- 随时取消：`cron cancel <job-id>`

---

### 2. 循环运行中

<p align="center">
  <img src="../../implementation/assets/impl-loop-2.png" alt="循环任务每分钟触发" width="100%">
</p>

任务每分钟触发一次，运行 `date` 命令并报告当前时间。每次迭代会触发异步的 **UserPromptSubmit** 和 **Stop** 钩子——与本仓库用于声音通知的钩子系统相同。

---

## ![如何使用](../../!/tags/how-to-use.svg)

```bash
$ claude
> /loop 1m "报告当前时间"
> /loop 5m /simplify
> /loop 10m "检查部署状态"
```

---

## ![如何实现](../../!/tags/how-to-implement.svg)

`/loop` 是 Claude Code 内置技能——无需任何设置。它在底层使用 cron 工具（`CronCreate`、`CronList`、`CronDelete`）来管理重复调度。
