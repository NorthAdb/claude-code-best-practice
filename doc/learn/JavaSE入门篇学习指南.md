## 入门篇整体定位

入门篇是整个 JavaSE 教程的第一阶段，面向**零基础**学习者。从计算机基础知识开始，逐步深入到 Java 语言的基本语法、数据类型、运算符、流程控制，最后通过三个面向过程编程实战巩固所学内容。UP 主的核心理念——**"基础不牢，地动山摇"** 贯穿始终。

## 模块一：计算机基础知识（P1-P4）

> 学习计算机的基本构成、操作系统和编程语言的分类，为后续学习 Java 做好前置知识储备。

| 分P | 标题 | 时长 | 内容概要 |
|-----|------|------|---------|
| **P1** | 课程介绍和注意事项 | ~15min | 课程定位、学习路线规划、Java SE 在整个 Java 体系中的位置 |
| **P2** | 计算机的世界 | ~9.5min | 计算机发展史、硬件组成（CPU、内存、硬盘、I/O 设备）、数据存储原理 |
| **P3** | 计算机操作系统 | ~7min | 操作系统的作用（资源管理、抽象硬件）、常见 OS（Windows/Linux/macOS） |
| **P4** | 计算机编程语言 | ~9min | 编程语言的分类（编译型 vs 解释型）、C/Python/Java 的对比、代码执行流程 |

**关键知识点：**
- 计算机五大核心组件：控制器、运算器、存储器、输入设备、输出设备
- 操作系统是硬件和应用软件之间的桥梁，管理 CPU 调度、内存分配、文件系统
- 编译型语言（如 C）：源码 → 编译器 → 机器码 → 直接执行
- 解释型语言（如 Python）：源码 → 解释器逐行执行，无需编译
- Java 是**编译+解释结合**：源码 → 字节码 → JVM 解释执行（跨平台的关键）

## 模块二：Java 初体验（P5-P7）

> 了解 Java 的历史和特性，搭建开发环境，安装并熟悉 IDEA 集成开发环境。

| 分P | 标题 | 时长 | 内容概要 |
|-----|------|------|---------|
| **P5** | 走进Java语言 | ~10min | Java 的历史、特性（跨平台、面向对象、自动内存管理）、JVM 原理简介 |
| **P6** | JDK安装与测试 | ~11.5min | JDK 下载安装、环境变量配置（PATH、JAVA_HOME）、javac/java 命令验证 |
| **P7** | IDEA安装与使用 | ~17min | IntelliJ IDEA 下载安装、创建第一个 Java 项目、项目结构（src/out）、运行 Hello World |

**关键知识点：**
- **JVM**（Java Virtual Machine）：Java 跨平台的基石，不同的 OS 有不同的 JVM 实现，但字节码统一
- **JDK vs JRE**：JDK = JRE + 开发工具（javac 等），JRE = JVM + 核心类库
- **环境变量**：`JAVA_HOME` 指向 JDK 安装目录，`PATH` 添加 `%JAVA_HOME%\bin`
- **IDEA 快捷键**：`psvm` + Tab 生成 main 方法，`sout` + Tab 生成 `System.out.println()`

## 模块三：Java 程序基础（P8-P12）

> 从第一个完整的 Java 程序开始，学习程序基本结构、注释、变量与常量，以及二进制在计算机中的表示。

| 分P | 标题 | 时长 | 内容概要 |
|-----|------|------|---------|
| **P8** | Java程序基本格式 | ~16min | 类定义、main 方法、包声明、import 语句，标准 Java 程序结构 |
| **P9** | 编写注释 | ~7.5min | 单行注释 `//`、多行注释 `/* */`、文档注释 `/** */`（javadoc）|
| **P10** | 变量与常量 | ~15min | 变量声明和初始化、命名规范（驼峰命名）、`final` 常量、字面量 |
| **P11** | 计算机中的二进制表示 | ~22min | 二进制/八进制/十六进制、原码/反码/补码、位权展开 |
| **P12** | 米奇试药问题 | ~4.5min | 经典算法思维题引入（为后续二进制和位运算打基础）|

**关键知识点：**
- Java 程序基本结构：
  ```java
  package com.example;         // 包声明
  import java.util.Scanner;    // 导入语句
  
  public class Main {          // 类定义
      public static void main(String[] args) {   // 入口方法
          System.out.println("Hello Java");      // 输出语句
      }
  }
  ```
- **标识符命名规则**：字母/数字/下划线/$ 组成，数字不能开头，区分大小写
- **变量**：`数据类型 变量名 = 初始值;`，如 `int age = 18;`
- **常量**：`final 数据类型 常量名 = 值;`，如 `final double PI = 3.14;`
- 二进制：计算机底层所有数据都用 0/1 表示
- **补码**：负数在计算机中以补码形式存储（原码取反 +1），这使得加减法可统一用加法电路实现

## 模块四：数据类型（P13-P17）

> 深入学习 Java 的四大基本数据类型系列——整型、浮点型、字符型和布尔型，以及 Java 10 的局部变量类型推断。

| 分P | 标题 | 时长 | 内容概要 |
|-----|------|------|---------|
| **P13** | 整数类型 | ~18min | `byte` / `short` / `int` / `long` 的取值范围和默认值、整数溢出问题 |
| **P14** | 浮点类型 | ~18.5min | `float` / `double` 精度、科学计数法、浮点数精度丢失问题 |
| **P15** | 字符类型 | ~19min | `char` 存储 Unicode 字符、转义字符、ASCII 码、字符与整数的关系 |
| **P16** | 布尔类型 | ~3min | `boolean` 只有 `true` / `false` 两个值，与 C 语言不同不能与整数互转 |
| **P17** | (Java 10) 局部变量类型推断 | ~2min | `var` 关键字：让编译器自动推断局部变量类型 |

**关键知识点：**
| 数据类型 | 关键字 | 占用字节 | 取值范围 |
|---------|--------|---------|---------|
| 字节型 | `byte` | 1 | -128 ~ 127 |
| 短整型 | `short` | 2 | -32,768 ~ 32,767 |
| 整型 | `int` | 4 | -2³¹ ~ 2³¹-1（约 ±21 亿）|
| 长整型 | `long` | 8 | -2⁶³ ~ 2⁶³-1 |
| 单精度浮点 | `float` | 4 | ±3.4E-38 ~ ±3.4E+38 |
| 双精度浮点 | `double` | 8 | ±1.7E-308 ~ ±1.7E+308 |
| 字符型 | `char` | 2 | 0 ~ 65,535（Unicode）|
| 布尔型 | `boolean` | 未定（JVM 实现相关）| `true` 或 `false` |

- **浮点数精度问题**：`0.1 + 0.2 != 0.3`——这是因为十进制小数转二进制时可能无限循环
- **Java 10+ `var`**：`var name = "Java";` → 编译器自动推断 `name` 为 `String` 类型，但只能用于局部变量

## 模块五：运算符（P18-P24）

> 全面掌握 Java 的各类运算符——赋值、算术、自增自减、位运算、关系、逻辑，以及运算符优先级。

| 分P | 标题 | 时长 | 内容概要 |
|-----|------|------|---------|
| **P18** | 赋值运算符 | ~5min | `=` 基本赋值、复合赋值（`+=`、`-=`、`*=` 等）|
| **P19** | 算数运算符 | ~34min | `+` `-` `*` `/` `%`、整数除法与浮点除法、类型转换（隐式/显式）|
| **P20** | 括号运算符 | ~15min | `()` 改变运算优先级、小括号的使用场景和最佳实践 |
| **P21** | 自增自减运算符 | ~10.5min | `++` / `--`、前缀式 vs 后缀式的区别（先取值后运算 vs 先运算后取值）|
| **P22** | 位运算符 | ~20min | `&` `|` `^` `~` `<<` `>>` `>>>`、位运算的实际应用场景 |
| **P23** | 关系运算符 | ~5min | `==` `!=` `>` `<` `>=` `<=`，返回布尔值 |
| **P24** | 逻辑运算符 | ~17min | `&&` `||` `!`、短路特性（`&&` 左侧为 false 则右侧不执行）|

**关键知识点：**
- **隐式类型转换**（自动）：`byte → short → int → long → float → double`（小转大）
- **显式类型转换**（强制）：`(目标类型) 值`，如 `(int) 3.14`，可能丢失精度
- **自增自减**：
  ```java
  int a = 5;
  int b = a++;   // b = 5, a = 6（先赋值，后自增）
  int c = ++a;   // c = 7, a = 7（先自增，后赋值）
  ```
- **短路逻辑**：`exp1 && exp2` → 若 `exp1` 为 false，直接返回 false 不计算 `exp2`
- **位运算优先级低于算术运算符**，用括号显式表达意图

## 模块六：代码结构与流程控制（P25-P31）

> 学习代码块作用域、选择结构（if/switch）、三种循环结构（for/while/do-while），以及 Java 9 交互式编程。

| 分P | 标题 | 时长 | 内容概要 |
|-----|------|------|---------|
| **P25** | 代码块和作用域 | ~7min | `{}` 作用域规则、局部变量的生命周期、嵌套作用域的变量屏蔽 |
| **P26** | 选择结构 | ~24min | `if` / `else if` / `else` 语法、三元运算符 `? :`、条件判断的最佳实践 |
| **P27** | (Java 14) switch表达式 | ~11.5min | 传统 `switch` 语句 vs Java 14+ 箭头 `->` switch 表达式、`yield` 返回值 |
| **P28** | 循环结构（一）| ~25min | `while` 循环、`do-while` 循环、循环三要素（初始化/条件/迭代）|
| **P29** | 循环结构（二）| ~13min | `for` 循环、`for-each` 增强 for 循环、循环嵌套 |
| **P30** | 循环结构（三）| ~7.5min | `break` / `continue` 关键字、带标签的 break/continue、死循环模式 |
| **P31** | (Java 9) 交互式编程 | ~4min | **JShell** 工具介绍：REPL 模式实时运行 Java 代码片段 |

**关键知识点：**
- **作用域**：`{}` 内声明的变量只在 `{}` 内有效，内层同名变量会屏蔽外层
- **if 语法**：
  ```java
  if (条件 1) {
      // 条件 1 为 true 时执行
  } else if (条件 2) {
      // 条件 1 为 false 且条件 2 为 true 时执行
  } else {
      // 全部为 false 时执行
  }
  ```
- **Java 14+ switch 表达式**：
  ```java
  String result = switch (score) {
      case 90, 95, 100 -> "优秀";
      case 80 -> "良好";
      default -> {
          yield "需努力";   // 多行语句用 yield 返回值
      }
  };
  ```
- **循环对比**：
  | 循环类型 | 执行时机 | 适用场景 |
  |---------|---------|---------|
  | `while(条件)` | 先判断，后执行 | 不确定循环次数 |
  | `do { } while(条件)` | 至少执行一次 | 至少需要执行一次 |
  | `for(初始化;条件;迭代)` | 先判断，后执行 | 确定循环次数 |
- **break**：跳出当前循环；**continue**：跳过本轮循环剩余语句进入下一轮
- JShell：在终端输入 `jshell` 启动，可直接执行 Java 语句、定义变量和方法，适合快速验证想法

## 模块七：面向过程编程实战（P32-P34）

> 通过三个经典编程题巩固前面所学的变量、运算符、循环等基础知识。

| 分P | 标题 | 时长 | 内容概要 |
|-----|------|------|---------|
| **P32** | 寻找水仙花数 | ~7min | 查找所有三位数的水仙花数（每位数字的立方和等于本身）|
| **P33** | 打印九九乘法表 | ~8min | 嵌套 for 循环打印梯形乘法表 |
| **P34** | 斐波那契数列 | ~5min | 输出指定项数的斐波那契数列（迭代实现）|

**实战代码思路：**

**1️⃣ 水仙花数**
```
思路：遍历 100~999
  对每个数：取出百位、十位、个位
  判断：百位³ + 十位³ + 个位³ == 原数
  成立则输出
```

**2️⃣ 九九乘法表**
```
思路：外层循环 1~9（行）
  内层循环 1~i（列）
  输出 i × j = 积
  每行结束换行
```

**3️⃣ 斐波那契数列**
```
思路：F(1)=1, F(2)=1, F(n)=F(n-1)+F(n-2)
  用变量迭代（a, b, sum）实现
  避免递归（性能问题后续再讲）
```

## 学习建议

1. **学习顺序**：建议严格按照 P1→P34 的顺序学习，前面的概念是后续内容的基础
2. **动手实践**：UP 主强调"基础不牢，地动山摇"，每节课后务必跟着敲代码
3. **重点章节**：
   - P8（程序格式）和 P10（变量常量）是后续所有代码的基石
   - P11（二进制）和 P15（字符类型）理解底层原理对后续学习很重要
   - P19（算术运算符/类型转换）时长最长，内容密集
   - P26-P30（选择/循环）是最常用的逻辑结构，务必熟练掌握
4. **JShell（P31）**：学习过程中随时用 JShell 快速验证代码片段
5. **实战题（P32-P34）**：不要直接看答案，先自己尝试实现
